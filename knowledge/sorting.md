# 排序算法 Sorting Algorithms

## 排序问题
排序是将一组数据按照升序或降序重新排列的过程。排序算法常用于数据预处理、查找优化和算法教学演示。

## 冒泡排序
冒泡排序通过相邻元素比较和交换，把较大的元素逐步移动到数组末尾。

特点：
- 思想简单，适合教学演示。
- 时间复杂度通常为 `O(n^2)`。
- 如果某一轮没有发生交换，可以提前结束。

Python 模板：
```python
def bubble_sort(nums):
    arr = nums[:]
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr
```

## 选择排序
选择排序每一轮从未排序区间中找到最小值，并放到当前起始位置。

特点：
- 交换次数较少。
- 时间复杂度稳定为 `O(n^2)`。
- 不适合大规模数据。

## 快速排序
快速排序选择一个基准值，将数组划分为小于等于基准和大于基准的两部分，再递归处理两边。

特点：
- 平均时间复杂度 `O(n log n)`。
- 最坏情况下可能退化为 `O(n^2)`。
- 常用于讲解分治思想。

## 常见错误
- 冒泡排序内层循环边界写错。
- 快速排序递归边界没有收缩，导致无限递归。
- 原地排序和返回新数组的行为混淆。
- 忘记考虑重复元素。

## 测试建议
- 已有序数组。
- 逆序数组。
- 包含重复元素。
- 空数组。
- 单元素数组。
